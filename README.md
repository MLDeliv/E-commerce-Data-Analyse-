# E-commerce-Data-Analyse
Maven Fuzzy Factory est une entreprise fictive de vente en ligne spécialisée dans les jouets en peluche pour enfants.   Comment Maven Fuzzy Factory peut-elle optimiser ses performances e-commerce entre 2012 et 2015 en identifiant les leviers de croissance du chiffre d'affaires, en améliorant les taux de conversion, et en maximisant la rentabilité?
Cette analyse poursuit les objectifs suivants :
  - Réaliser un diagnostic complet des ventes globales (2012-2015) avec identification des tendances de croissance
  - Évaluer les performances par canal marketing UTM et orienter les investissements publicitaires
  - Analyser le tunnel de conversion end-to-end et identifier les étapes à fort taux d'abandon
  - Comparer les comportements mobile vs desktop pour prioriser les efforts d'optimisation UX
  - Formuler des recommandations concrètes et actionnables pour la direction commerciale
